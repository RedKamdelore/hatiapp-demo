# HatiApp

Веб-приложение для планирования волонтерских смен. Поддерживает расписание по направлениям, запись на смены, учет посещаемости, чат между участниками, стену объявлений и обмен сменами.

## Возможности

- Расписание смен по направлениям и дням
- Запись на смены и отмена записи
- Роли: волонтер, руководитель направления, администратор
- QR-сканирование для отметки посещаемости
- Чат между любыми пользователями, пересылка сообщений
- Стена объявлений: посты, комментарии, опросы, реакции
- Обмен сменами через чат: предложение, принятие, отклонение, отмена
- Блокировка дней, массовые действия над пользователями
- Темная тема и PWA-режим с офлайн-кешированием
- Печатная версия расписания

## Технологии

- Python 3.11
- FastAPI
- SQLAlchemy + SQLite
- Jinja2 + Tailwind CSS
- Alembic для миграций
- APScheduler для фоновых задач

## Установка

```bash
python3 -m venv venv
source venv/bin/activate  # Windows: .\venv\Scripts\activate
pip install -r requirements.txt
alembic upgrade head
```

## Запуск

```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

После запуска приложение доступно по адресу `http://localhost:8000`.

## Тестовые данные

Перейдите по адресу `/seed`, чтобы создать тестовых пользователей:

- `admin` / `admin` — администратор
- `vol1` / `vol123` — волонтер

## Тесты

```bash
pytest
```

Последний прогон: 140 тестов проходят, 1 пропущен.

## Миграции

```bash
alembic revision --autogenerate -m "описание изменений"
alembic upgrade head
```
